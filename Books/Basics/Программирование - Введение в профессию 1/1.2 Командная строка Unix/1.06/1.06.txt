Задача:

Что означают права доступа к обычному файлу, заданные следующими числами?
a) 0700
b) 0440
c) 0511
d) 0660
e) 0644
f) 0555

--------

Ответ:

a) все разрешено для владельца, все запрещено для групп и прочих пользователей
b) только чтение для владельца и групп, все запрещено для прочих пользователей
c) чтение и исполнение для владельца, только выполнение для групп и прочих пользователей
d) чтение и запись для владельца и групп, все запрещено для прочих пользователей
e) чтение и запись для владельца, только чтение для групп и прочих пользователей
f) чтение и выполнение для всех
